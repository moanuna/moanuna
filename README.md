## 📝 About ME
### 👋 보이지 않는 곳에서 대시보드를 만들며 인프라의 안정성을 다지는 엔지니어입니다.

 * 대시보드를 통한 상시 관제 체계를 구축하여 안정적인 서비스를 만드는 것을 좋아합니다.
 * 애매하게 터지는 에러의 근본적인 원인을 분석하고 선명하게 해결하는 과정을 중요하게 생각합니다.
 * 프로젝트의 요구사항과 팀의 니즈를 인프라 기반으로 구현해 나가는 개발자입니다.
